**About**

This project contains everything you need to run your app locally.

**Edit the code in your local development environment**

Any change pushed to the repo will also be reflected in the Base44 Builder.

**Prerequisites:** 

1. Clone the repository using the project's Git URL 
2. Navigate to the project directory
3. Install dependencies: `npm install`
4. Create an `.env.local` file and set the right environment variables

```
VITE_BASE44_APP_ID=your_app_id
VITE_BASE44_APP_BASE_URL=https://your-app-slug.base44.app
# Optional: only when using a local Base44 backend (`base44 dev`)
# VITE_BASE44_SERVER_URL=http://127.0.0.1:8787
```

The SDK client omits `serverUrl` in production so requests go to `https://base44.app` (current SDK default). Do not set an empty `serverUrl` in code — it breaks API base URLs.

Run the app: `npm run dev`

**Publish your changes**

Open [Base44.com](http://Base44.com) and click on Publish.
